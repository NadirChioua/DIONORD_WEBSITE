# Images à placer dans le projet DIONORD

## Logo (OBLIGATOIRE)
- `public/images/logo.png` — Le logo DIONORD (engrenage + bâtiment + texte)

## Réalisations (photos réelles de l'entreprise)
Placer dans `public/images/realisations/` :

| Fichier | Description | Utilisé dans |
|---------|-------------|--------------|
| `garde-corps-inox.jpg` | Garde-corps inox d'escalier intérieur | Services alu, Accueil |
| `cloisons-vitrees.jpg` | Cloisons vitrées de bureau (alu/verre) | Services alu, Accueil |
| `paroi-douche.jpg` | Paroi de douche aluminium | Services alu, Accueil |
| `charpente-construction.jpg` | Charpente métallique vue extérieure + grue | Hero slider, Construction |
| `charpente-interieur.jpg` | Charpente métallique vue intérieure | Hero slider, Construction |
| `fabrication-atelier.jpg` | Fabrication structure métallique en atelier | Atelier, Machines |

## Formats recommandés
- Format : JPG ou WebP
- Dimensions : 1200×800px minimum pour les grandes images
- Qualité : 80-85% (optimisé taille/qualité)
- Poids : max 500KB par image (Next.js optimise automatiquement)

## Une fois les images placées
Le site est prêt à démarrer avec :
```bash
npm run dev
```
